Frequently asked questions

Have more questions? See full FAQ.

  • What is USSD?

    Unstructured Supplementary Service Data (USSD) is a way of communicating with your mobile network or a web-based service using short-codes that start with * and end with #. It can be used to provide various online services without using the internet or installing any apps on your phone.

    What are USSD sessions?

    Whenever a USSD code is dialed, a session is created that lasts for the duration of the interaction.

  • What is a USSD short code?

    A short code is a special phone number that is shorter than a regular phone number and is used to facilitate communication between mobile devices and the telecom operator.

    Is Uzo cloud based?

    Yes, Uzo is cloud based. Enterprises can request a self-hosted version of Uzo for private servers if needed.

  • What are workload units?

    Workload units represent the basic unit of processing for an Uzo App, required for each execution of your application.

    What is a dedicated USSD code?

    A dedicated USSD code is a short number assigned and used by only one enterprise or service, such as *123#.
