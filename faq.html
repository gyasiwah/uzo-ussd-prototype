Frequent Asked Questions
Unstructured Supplementary Service Data (USSD) is a way of communicating with your mobile network or a web-based service using short-codes that start with * and end with #. It can be used to provide various online services without using the internet or installing any apps on your phone.
A short code is a special phone number that is shorter than a regular phone number and is used to facilitate communication between mobile devices and the telecom operator. Short codes are designed to be easier to remember and use than regular phone numbers.
Workload units represent the basic unit of processing for an Uzo App. For each execution of your application, a specific number of workload units are required for processing. Workload units are required by applications in deployed state to ensure accessibility.
Whenever a USSD code is dialed, a session is created that lasts for the duration of the interaction.
Yes, Uzo is cloud based. However, enterprises who wish to host their USSD connections on their private servers can request for a self-hosted version of Uzo. The self-hosted version of Uzo gives users full control while maintaining all the value Uzo provides as a USSD gateway and a USSD code management platform.
A dedicated USSD code is a short number that is assigned and used by only one enterprise or service. It has only one segment, such as *123#. A dedicated USSD code allows enterprises to have full control and flexibility over their USSD services. It also gives enterprises access to a wide range of extensions such as *123*34#.
To register a dedicated USSD short-code, register and login to your Uzo portal. Click on "Dedicated Codes" and then "Purchase Dedicated Code". Follow the steps to complete the process.
USSD code activation on all networks takes between 2 to 5 weeks.
A USSD extension is a variation of the dedicated USSD code that gives users the ability to have multiple applications riding on the same dedicated USSD code. For example, a dedicated code *1234# can have extensions such as *1234*1#, *1234*5#, *1234*12*54# etc. linked to completely different applications running concurrently with your main application.
Login to the Uzo management portal, click on "USSD Extension" on the left panel and proceed to create a new extension for any of your dedicated codes.
You can create as many USSD extensions as you want on your dedicated USSD code at no additional charge.
Uzo currently operates in Ghana only. However, expansion is ongoing to ensure that our Uzoers get access to USSD codes in other countries. Uzoers will be notified whenever operation in a new country begins.
